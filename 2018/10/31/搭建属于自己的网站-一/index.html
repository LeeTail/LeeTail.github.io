搭建属于自己的网站(一)

发表于 2018-10-31

这次分享一下搭建网站的一些简略步骤。我先在这篇博文分享一下利用iis搭建网站的经过，在后续时间里，
我有空会分享一下phpstudy搭建网站以及centos搭建网站的一些坑。
咳咳，进入正题。
iis是Internet Information Services的缩写,意为互联网信息服务,是由微软公司提供的基于运行MicrosoftWindows的互联网基本服务。最初是Windows NT版本的可选包,随后内置在Windows 2000、Windows XP Professional和Windows Server 2003一起发行，
但在Windows XP Home版本上并没有IIS。IIS是一种Web（网页）服务组件，其中包括Web服务器、FTP服务器、NNTP服务器和SMTP服务器，
分别用于网页浏览、文件传输、新闻服务和邮件发送等方面，它使得在网络（包括互联网和局域网）上发布信息成了一件很容易的事。（摘自百度百科）
简单点来说，iis就类似于一种容器(web容器)用来存放我们的网页文件资源。
(⊙o⊙)…首先我们要准备好iis，这里有两种选择，一是利用自己真机的win10系统自带的iis（并不建议），我们搭网站
只是用来实验，完全没有必要在真机上实践。二是利用微软的服务器系统，有2003,2008,2012以及2016这几个版本，
我个人用的是2012的，看自己选择。 至于镜像的话，我就不提供了，自行百度(windows server 2012 r2)，版本自己改就行，
进官网下载就行，网上也有安装windows server 2012 r2的教程。下载好镜像后，关于虚拟机，有两种，一个是VMWare（收费的），另一个
是VistualBox（orale的，免费的），也可以用破解版，自行百度下载即可以。
当虚拟机安装好后，创建一个windows server 2012 r2的虚拟机，然后打开，配好自己的系统后，系统会自动打开一个服务器管理器。

然后点击管理，在点击添加角色和功能向导

然后下一步，直到服务器角色

然后点击自己要安装的角色，这里我选择了iis，也可以多选其它，比如我选了DNS，也可以选FTP等等。然后下一步

然后进入功能选择，默认就行。下一步
